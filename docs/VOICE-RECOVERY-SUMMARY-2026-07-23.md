# Voice recovery summary — superseded archive

This document used to contain a long reconstruction of historical voice
experiments. It mixed useful evidence with fixed speeds, fixed pause ladders,
line-length recipes, retired cast routes, and processing profiles. Those values
are not safe authoring defaults and are not runtime truth.

The detailed reconstruction was removed from the working tree to stop it from
being copied back into code. Git history remains the archive.

For current work read, in order:

1. `config/voice/personas.toml`
2. `config/voice/gains.json`
3. `dan/voice/policy.py`
4. `MUST-READ-GLOS-PROZODIA.md`
5. `docs/PROZODIA.md`

Historical listening evidence can motivate a new controlled test. It cannot
silently create a persona, profile, effect, tempo, pause, or scene template.
