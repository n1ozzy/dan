Wartościowa lekcja z odsłuchów historycznych jest jakościowa, nie liczbowa:

- pełna myśl i kontekst brzmią lepiej niż mechaniczne szatkowanie;
- filtr i zmiana barwy nie zastępują aktorstwa;
- postać musi reagować, a nie czytać dokument z doklejonym przekleństwem;
- enqueue, manifest i zielony test nie dowodzą naturalności;
- o wyniku decyduje rzeczywisty odsłuch.

usunięte z tego pliku celowo, ponieważ wielokrotnie wracały do aktywnego kodu
